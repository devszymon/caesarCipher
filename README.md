# Caesar cipher

Caesar cipher implementation in python.

## Installation
- Install python 3.11
- Clone this repository to your desktop.
- Navigate to the top level of the directory
- Run facade.py


## Features

- Lightweight
- Encryption and Decryption from plain text
- Json support for data
- Logging function
- Save results to file
